# Element Technology - Preview multi-pages

Version de présentation du site en pages séparées :

- `index.html` : Accueil
- `enr.html` : ENR / photovoltaïque
- `its.html` : ITS / régulation du trafic
- `certification.html` : Certification RGE PV500
- `contact.html` : Contact
- `styles.css` : styles communs
- `assets/` : médias récupérés depuis le site actuel

## Publication GitHub Pages

Uploader tout le contenu de ce dossier à la racine du dépôt GitHub.

Ensuite, dans GitHub :

1. `Settings`
2. `Pages`
3. Source : `Deploy from a branch`
4. Branch : `main`
5. Folder : `/root`
6. `Save`

Le site sera disponible à l'adresse GitHub Pages du dépôt.
